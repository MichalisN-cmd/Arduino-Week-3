περιγραφή
Το project αυτό χρησιμοποιεί τον υπερηχητικό αισθητήρα HC-SR04 για τη μέτρηση αποστάσεων και δίνει οπτικές και ηχητικές ενδείξεις μέσω **LED** και buzzer.  
Καθώς ένα αντικείμενο πλησιάζει τον αισθητήρα, ενεργοποιούνται σταδιακά τα LED και οι βομβητές (buzzers), προειδοποιώντας τον χρήστη για την εγγύτητα.

 Υλικά
- Arduino Uno ή συμβατός μικροελεγκτής  
- Υπερηχητικός αισθητήρας **HC-SR04**  
- 3 LED (πράσινο, κίτρινο, κόκκινο)  
- 3 Αντιστάσεις 220Ω  
- 2 Buzzer  
- Breadboard και καλώδια σύνδεσης

 Συνδεσμολογία
| Εξάρτημα | Arduino Pin |
| HC-SR04 Trig | 9 |
| HC-SR04 Echo | 10 |
| Πράσινο LED | A2 |
| Κίτρινο LED | A3 |
| Κόκκινο LED | A4 |
| Buzzer 1 | 12 |
| Buzzer 2 | 8 |
| GND | Κοινή γείωση για όλα τα εξαρτήματα |

Κάθε LED συνδέεται μέσω αντίστασης 220Ω στη γείωση (GND).

 Λειτουργία
- Απόσταση **> 100 cm** → Ανάβει **μόνο το πράσινο LED** (αντικείμενο μακριά).  
- Απόσταση **50–100 cm** → Ανάβει **το κίτρινο LED** (αντικείμενο πλησιάζει).  
- Απόσταση **20–50 cm** → Ανάβει **το κόκκινο LED** και **οι δύο buzzers** εκπέμπουν ήχο συνεχώς.  
- Απόσταση **< 20 cm** → Το κόκκινο LED **αναβοσβήνει** και οι δύο buzzers **παίζουν δύο διαφορετικούς τόνους** (440 Hz και 523 Hz) εναλλάξ, δημιουργώντας προειδοποιητικό ήχο.
